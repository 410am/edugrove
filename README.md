## 📕 설명


Web RTC를 이용한 화상 채팅 서비스를 만들어 보았습니다.
<br>이름에서도 알 수 있듯 교육용 화상수업 서비스를 컨셉으로 설정했으며, 그에 알맞은 단순하고 큼직한 UI로 디자인 했습니다.
<br>React, TypeScript를 사용하여 프론트를 진행하였으며, 서버에는 express가 사용되었습니다.
